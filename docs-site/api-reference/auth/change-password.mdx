---
title: "Change Password"
api: "POST /v1/auth/change-password"
description: "Change the authenticated merchant's account password"
---

<Note>
  This endpoint requires a valid Bearer token in the `Authorization` header.
</Note>

## Request Body

<ParamField body="currentPassword" type="string" required>
  The merchant's current password.
</ParamField>

<ParamField body="newPassword" type="string" required>
  The new password. Minimum 8 characters, must include uppercase, lowercase, and a number.
</ParamField>

### Example Request

```json
{
  "currentPassword": "SecureP@ss123",
  "newPassword": "NewSecureP@ss456"
}
```

### Example Response (200)

```json
{
  "message": "Password changed successfully"
}
```

### Error Responses

```json 401 Wrong Password
{
  "error": {
    "code": "UNAUTHORIZED",
    "message": "Current password is incorrect"
  }
}
```

```json 400 Validation Error
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "New password must be at least 8 characters"
  }
}
```
